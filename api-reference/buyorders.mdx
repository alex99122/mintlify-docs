## Create buy order

```
POST /item/buyorder
```

**Permission:** Connected + API

Creates a buy order for a specific item at a specified price.

### Body Parameters

<ParamField body="itemId" type="integer" required>
  Item ID for which to create a buy order
</ParamField>

<ParamField body="price" type="integer" required>
  Buy order price in cents
</ParamField>

<ParamField body="quantity" type="integer" required>
  Number of items to buy (default: 1)
</ParamField>

### Request Example

```json
{
  "itemId": 12345,
  "price": 15000,
  "quantity": 3
}
```

### Response

```json Success
{
  "err": false,
  "success": true,
  "content": {
    "details": "Inserted"
  }
}
```

### Error Responses

| Status | Message | Description |
|--------|---------|-------------|
| `error` | Insufficient balance | User doesn't have enough balance to create the buy order |
| `error` | Price too low | Buy order price is below minimum threshold |
| `error` | Item not found | Invalid item ID |
| `error` | Buy order limit reached | User has reached the maximum number of buy orders |

<Info>
  The total price (price Ã— quantity) will be reserved from your balance when creating a buy order.
</Info>

## Update buy order

```
POST /item/buyorder/update
```

**Permission:** Connected + API

Updates an existing buy order's price or quantity.

### Body Parameters

<ParamField body="buyOrderId" type="integer" required>
  Buy order ID to update
</ParamField>

<ParamField body="price" type="integer">
  New price in cents (optional)
</ParamField>

<ParamField body="quantity" type="integer">
  New quantity (optional)
</ParamField>

### Request Example

```json
{
  "buyOrderId": 98765,
  "price": 16000,
  "quantity": 5
}
```

### Response

```json Success
{
  "err": false,
  "success": true,
  "content": "Buy order updated"
}
```

<Warning>
  Updating a buy order may require additional balance if increasing price or quantity. The difference will be reserved from your available balance.
</Warning>

## Remove buy order

```
POST /item/buyorder/remove
```

**Permission:** Connected + API

Cancels and removes a buy order, releasing the reserved balance.

### Body Parameters

<ParamField body="buyOrderId" type="integer" required>
  Buy order ID to remove
</ParamField>

### Request Example

```json
{
  "buyOrderId": 98765
}
```

### Response

```json Success
{
  "err": false,
  "success": true,
  "content": {
    "details": "Removed"
  }
}
```

## Get user's buy orders for an item

```
GET /user/buyorder/{item}
```

**Permission:** Connected + API

<ParamField path="item" type="string" required>
  Item identifier (numeric ID)
</ParamField>

Returns all active buy orders for a specific item created by the authenticated user.

### Response

```json
{
  "err": false,
  "success": true,
  "content": {
    "informations": {
      "id": 98765,
      "steamid": "76561198000000000",
      "itemid": 12345,
      "price": 15000,
      "amount": 3,
      "timestamp": "1706745600"
    }
  }
}
```

### Buy Order Fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | integer | Buy order ID |
| `steamid` | string | User's Steam ID |
| `itemid` | integer | Item ID |
| `price` | integer | Buy order price in cents |
| `amount` | integer | Quantity of items to buy |
| `timestamp` | string | Unix timestamp of creation |

## Get all user's buy orders

```
GET /user/getBuyorder
```

**Permission:** Connected + API

Returns all active buy orders for the authenticated user across all items.

### Query Parameters

<ParamField query="page" type="integer" default="0">
  Page number for pagination
</ParamField>

<ParamField query="perpage" type="integer" default="20">
  Number of results per page (max 50)
</ParamField>

### Response

```json
{
  "err": false,
  "success": true,
  "content": {
    "values": [
      {
        "id": 98765,
        "steamid": "76561198000000000",
        "itemid": 12345,
        "price": 15000,
        "amount": 3,
        "timestamp": "1706745600",
        "name": "Team Captain",
        "effect": "Burning Flames",
        "url": "unusual_team_captain",
        "game": 440,
        "quality": "Unusual",
        "image": "https://...",
        "type": "Hat",
        "craftable": 1
      }
    ],
    "count": 15
  }
}
```

<Info>
  Buy orders are automatically filled when matching items are listed at or below your buy order price. You'll receive a notification when a buy order is filled.
</Info>

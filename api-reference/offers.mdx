## Get my offers

```
GET /offers/my
```

**Permission:** Connected + API

Returns trade offers that the authenticated user has sent to other users.

### Query Parameters

<ParamField query="status" type="integer">
  Filter by offer status
</ParamField>

### Response

Same format as received offers, but from the perspective of the sender. Note: Currently returns an empty string instead of the offers array.

## Create offer

```
POST /offers/create
```

**Permission:** Connected + API

Creates a new trade offer to purchase items from another user.

### Body Parameters

<ParamField body="items" type="array" required>
  Array of backpack item IDs to purchase
</ParamField>

<ParamField body="totalPrice" type="integer">
  Optional: Total price validation (in cents)
</ParamField>

### Request Example

```json
{
  "items": [12345, 12346],
  "totalPrice": 250000
}
```

### Response

```json Success
{
  "status": "success",
  "data": {
    "message": "Offer created successfully"
  }
}
```

### Error Responses

| Status | Message | Description |
|--------|---------|-------------|
| `error` | Insufficient balance | User doesn't have enough balance to create the offer |
| `error` | Items not available | One or more items are no longer available for purchase |
| `error` | Cannot buy your own items | User attempted to create an offer for their own items |

<Warning>
  Offers expire after 48 hours if not accepted or declined. The buyer's balance is reserved during this time.
</Warning>

## Remove/Cancel offer

```
POST /offers/remove
```

**Permission:** Connected + API

Cancels a sent offer (only available for offers you created).

### Body Parameters

<ParamField body="offerId" type="integer" required>
  The offer ID to cancel
</ParamField>

### Request Example

```json
{
  "offerId": 12345
}
```

### Response

```json Success
{
  "status": "success",
  "data": {
    "message": "Offer removed successfully"
  }
}
```

<Info>
  Cancelling an offer immediately releases the reserved balance back to the buyer's account.
</Info>

## Get received offers

```
GET /offers/received
```

**Permission:** Connected + API

Returns trade offers that the authenticated user has received from other users.

### Query Parameters

<ParamField query="status" type="integer">
  Filter by offer status (0 = active, 1 = accepted, 2 = declined, 3 = cancelled)
</ParamField>

### Response

```json
{
  "status": "success",
  "data": [
    {
      "id": 12345,
      "buyer": "76561198000000000",
      "seller": "76561198111111111",
      "items": [
        {
          "backpackId": 98765,
          "name": "Unusual Team Captain",
          "price": 150000
        }
      ],
      "totalPrice": 150000,
      "status": 0,
      "timestamp": "1706745600",
      "expiresAt": "1706832000"
    }
  ]
}
```

## Get my offers

```
GET /offers/my
```

**Permission:** Connected + API

Returns trade offers that the authenticated user has sent to other users.

### Query Parameters

<ParamField query="status" type="integer">
  Filter by offer status
</ParamField>

### Response

Same format as received offers, but from the perspective of the sender.

## Create offer

```
POST /offers/create
```

**Permission:** Connected + API

Creates a new trade offer to purchase items from another user.

### Body Parameters

<ParamField body="items" type="array" required>
  Array of backpack item IDs to purchase
</ParamField>

<ParamField body="totalPrice" type="integer">
  Optional: Total price validation (in cents)
</ParamField>

### Request Example

```json
{
  "items": [12345, 12346],
  "totalPrice": 250000
}
```

### Response

```json Success
{
  "status": "success",
  "data": {
    "offerId": 67890,
    "items": [
      {
        "backpackId": 12345,
        "price": 150000
      },
      {
        "backpackId": 12346,
        "price": 100000
      }
    ],
    "totalPrice": 250000,
    "expiresAt": "1706832000"
  }
}
```

### Error Responses

| Status | Message | Description |
|--------|---------|-------------|
| `error` | Insufficient balance | User doesn't have enough balance to create the offer |
| `error` | Items not available | One or more items are no longer available for purchase |
| `error` | Cannot buy your own items | User attempted to create an offer for their own items |

<Warning>
  Offers expire after 48 hours if not accepted or declined. The buyer's balance is reserved during this time.
</Warning>

## Accept offer

```
POST /offers/accept
```

**Permission:** Connected + API

Accepts a received trade offer.

### Body Parameters

<ParamField body="offerId" type="integer" required>
  The offer ID to accept
</ParamField>

### Request Example

```json
{
  "offerId": 12345
}
```

### Response

```json Success
{
  "status": "success",
  "data": {
    "offerId": 12345,
    "tradeId": "987654321",
    "message": "Offer accepted, trade initiated"
  }
}
```

## Remove/Cancel offer

```
POST /offers/remove
```

**Permission:** Connected + API

Cancels a sent offer (only available for offers you created).

### Body Parameters

<ParamField body="offerId" type="integer" required>
  The offer ID to cancel
</ParamField>

### Request Example

```json
{
  "offerId": 12345
}
```

### Response

```json Success
{
  "status": "success",
  "data": {
    "offerId": 12345,
    "message": "Offer cancelled",
    "refundedBalance": 150000
  }
}
```

<Info>
  Cancelling an offer immediately releases the reserved balance back to the buyer's account.
</Info>

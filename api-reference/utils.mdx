## Check trade URL

```
GET /utils/tradeurlCheck
```

**Permission:** Connected + API

Validates a Steam trade offer URL format and checks if it's accessible.

### Query Parameters

<ParamField query="tradeUrl" type="string" required>
  Steam trade offer URL to validate
</ParamField>

### Example

```
GET /utils/tradeurlCheck?tradeUrl=https://steamcommunity.com/tradeoffer/new/?partner=123456789&token=AbCdEfGh
```

### Response

```json Valid URL
{
  "status": "success",
  "data": {
    "valid": true,
    "partnerId": "123456789",
    "token": "AbCdEfGh",
    "steamId": "76561198083722517",
    "message": "Trade URL is valid"
  }
}
```

```json Invalid URL
{
  "status": "error",
  "data": {
    "valid": false,
    "message": "Invalid trade URL format"
  }
}
```

### Validation Checks

The endpoint performs the following validations:

1. **Format Check:** Verifies URL starts with `https://steamcommunity.com/tradeoffer/new/`
2. **Parameter Check:** Ensures `partner` and `token` parameters are present
3. **Token Validation:** Validates token format (8 characters, alphanumeric)
4. **Steam ID Conversion:** Converts partner ID to full Steam ID64

<Info>
  Use this endpoint before saving a trade URL to ensure it's valid and properly formatted.
</Info>

## Get bots information

```
GET /utils/bots
```

**Permission:** Public

Returns information about all Mannco.store trading bots, including their status, capacity, and trade URLs.

### Response

```json
{
  "status": "success",
  "data": {
    "bots": [
      {
        "steamId": "76561198000000001",
        "name": "Mannco Bot #1",
        "avatar": "https://avatars.steamstatic.com/...",
        "online": true,
        "games": [440, 730],
        "capacity": 1000,
        "currentItems": 756,
        "availableSlots": 244,
        "tradeUrl": "https://steamcommunity.com/tradeoffer/new/?partner=...",
        "lastSeen": "1706745600"
      },
      {
        "steamId": "76561198000000002",
        "name": "Mannco Bot #2",
        "avatar": "https://avatars.steamstatic.com/...",
        "online": true,
        "games": [440],
        "capacity": 1000,
        "currentItems": 823,
        "availableSlots": 177,
        "tradeUrl": "https://steamcommunity.com/tradeoffer/new/?partner=...",
        "lastSeen": "1706745580"
      }
    ],
    "totalBots": 5,
    "onlineBots": 4,
    "totalCapacity": 5000,
    "totalItems": 3421,
    "availableSlots": 1579
  }
}
```

### Bot Fields

| Field | Type | Description |
|-------|------|-------------|
| `steamId` | string | Bot's Steam ID |
| `name` | string | Bot display name |
| `avatar` | string | Bot's Steam avatar URL |
| `online` | boolean | Whether bot is currently online |
| `games` | array | Game IDs the bot handles |
| `capacity` | integer | Maximum items bot can hold |
| `currentItems` | integer | Current number of items in bot inventory |
| `availableSlots` | integer | Remaining capacity |
| `tradeUrl` | string | Bot's trade offer URL |
| `lastSeen` | string | Unix timestamp of last online check |

<Info>
  Bot information is cached for 5 minutes. The system automatically distributes items across bots based on availability and game.
</Info>
